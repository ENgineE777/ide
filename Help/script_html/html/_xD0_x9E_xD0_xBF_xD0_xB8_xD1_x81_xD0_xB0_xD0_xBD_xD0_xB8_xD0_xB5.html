формата LabelRich

Форматирование текста поддерживается виджетом NGUILabelRich скриптовой функцией SetText.

  • Данный формат основан на добовление в текст тэгов.
  • Упровляющие символы "# { }".
  • Для отображения данных символов в текст нужно записать "\# \{ \}".
  • Тэги начинаются с сивола # и заканчивается разделителем пробелом.

    Пример:
    skin::NGUILabelRich0.SetText("#fonttbl{#fn{dejavu serif}#fi#sh#r200#g200#b200}#colortbl{#r0#g0#b0;#r255#g0#b0;#g255} #tab2#fi#g255 sdg#fi0 Abc#f0 Abcsdg #b255#lnk{test1} yhe wrt eut #lnk0#clr1#fz26 FTY #algC RVW fghr #algL#fz20 juhdsf #scl2 sdufyh #scl1#algR ewu dgh #ln iurtg rh rfr4");

    Список тэгов:
  • #tab - Вставляет пробелы в текст, по умолчанию вставляется 4 пробела. Кол-во пробелов можно указать цифрой после имени тэга #tab2
  • #ln - Переход на новую строчку.
  • #algL - Выравнивание текста Align left Aвтоматически переводит на новую строчкку
  • #algR - Выравнивание текста Align right Aвтоматически переводит на новую строчкку
  • #algC - Выравнивание текста Align center Aвтоматически переводит на новую строчкку
  • #r #b #g #a- Задают цвет и прозоачность текста.
    Пример: skin::NGUILabelRich0.SetText("#r255 ABC#g255 DEF"); Tекст ABC - рисуется зеленым, текст DEF рисуется зеленым.
  • #sh - Включение отрисовки тени текста
  • #sh0 - Выключение отрисовки тени текста
  • #shr #shb #shg #sha- Задают цвет и прозоачность тени текста.
  • #fn{имя} - Установка имени фонта для текста
  • #fz - Установка высоты фонта текста
    Пример: skin::NGUILabelRich0.SetText("#fz20 ABC#fz26 DEF");
  • #fi - Включение режима italic для фонта
  • #fi0 - Выключение режима italic для фонта
  • #fb - Включение режима bold для фонта
  • #fb0 - Выключение режима bold для фонта
  • #gap - Установка межстрочного интервала для фонта.
    Пример: skin::NGUILabelRich0.SetText("#gap0.2 ABC#gap0.9 DEF");
  • #lnk{имя} - Начинает кликабельную зону текста. В скобка имя функции которая будет вызываться при клике по тексту.
  • #lnk{имя,index} - Начинает кликабельную зону текста. В скобка имя функции которая будет вызываться при клике по тексту и параметр для этой функци , типа int .
  • #lnk0 - Заканчивает кликабельную зону текста.

    Список специальных тэгов:
  • #colortbl{...} - Задает таблицу цветов. Каждый цвет разделяется символом ;
    Пример: skin::NGUILabelRich0.SetText("#colortbl{#r255;#g255}");
  • #clr - Устанавливает табличный цвет текст. Индексация в таблице начинается с 0.
    Пример: skin::NGUILabelRich0.SetText("#clr0 ABC#clr1 DEF");
  • #shclr - Устанавливает табличный цвет тени текст. Индексация в таблице начинается с 0.
    Пример: skin::NGUILabelRich0.SetText("#shclr0 ABC#shclr1 DEF");
  • #fonttbl{...} - Задает таблицу фонтов. Каждый фонт разделяется символом; Внутри символов {} можно исполбзовать тэги:
    #fn{имя} #r #b #g #a #shr #shb #shg #sha #sh #fi #fb #fz #gap
    Пример:
    skin::NGUILabelRich0.SetText("#fonttbl{#fn{dejavu serif}#fi#sh#r200#g200#b200;#fn{dejavu serif}#fi#fb#fz30}
  • #f - Устанавливает табличный цвет текст. Индексация в таблице начинается с 0.
    Пример: skin::NGUILabelRich0.SetText("#f0 ABC#f1 DEF");
